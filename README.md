## Description

The application is a sample Nest.js application that tests an RDS connection through AWS Secrets Manager and an S3 connection through a VPC endpoint.

## Installation

```bash
$ npm install
```

## Running the app

> Edit `.env` file

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run deploy
```
